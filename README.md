# Калькулятор водохозяйственного баланса

Полнофункциональное React/TypeScript приложение для расчета фактического и прогнозного ВХБ по бассейнам, ВХУ, створам, месяцам и сценариям водности.

## Возможности
- помесячный ввод прихода и расхода в млн м³ и м³/с;
- автоматический перевод м³/с в млн м³ по числу секунд в месяце;
- распределение годового значения по месяцам;
- расчет свободного стока, дефицита, обеспеченности и невязки;
- передача свободного стока в нижележащий ВХУ;
- расчет водохранилищ с контролем мертвого и полного объема;
- приоритеты обеспечения при дефиците;
- проверки исходных данных;
- импорт/экспорт Excel через SheetJS;
- PDF через печать браузера;
- хранение в Supabase/PostgreSQL или localStorage при отсутствии Supabase.

## Запуск
```bash
npm install
npm run dev
```

## Supabase
1. Создайте проект Supabase.
2. Выполните SQL из `schema.sql`.
3. Скопируйте `.env.example` в `.env` и заполните:
```bash
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```

Если переменные не заданы, приложение работает локально через `localStorage`.

## Важное замечание
Демонстрационные данные подписаны как условные и не являются официальными значениями Есильского или другого бассейна Казахстана.
